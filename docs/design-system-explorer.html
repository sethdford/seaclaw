Design Principles

Neutral Stage

The UI is a quiet stage. Warm charcoal backgrounds recede. Content and data are the protagonists. No element competes for attention unless it earns it.

Cinematic Light

Every shadow is warm. Every highlight is cream, not white. Glass refracts. Gradients breathe. Film grain adds texture. Nothing is flat.

Spring Physics

Every motion follows spring curves. Micro for instant feedback, Standard for navigation, Expressive for delight, Dramatic for emphasis. Never linear.
