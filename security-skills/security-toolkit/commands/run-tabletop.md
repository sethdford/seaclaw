# Run Tabletop Command

Chain these steps:

1. Use `incident-response-plan` to review IR plan and identify scenarios to test
2. Use `red-team-exercise` to design tabletop scenario with realistic attack simulation
3. Execute tabletop with team participation and observation
4. Facilitate after-action review and document lessons learned

Deliverables:

- Tabletop scenario and injects (simulated events)
- Participant roles and responsibilities
- Facilitator guide and timeline
- Observation logs and incident timeline during exercise
- After-action report with gaps identified
- Recommendations for IR plan improvements and team training

After completion, suggest follow-up commands: `launch-champion-program`, `design-training`.
