# Prepare Audit Command

Chain these steps:

1. Use `audit-preparation` to develop audit readiness plan
2. Use `compliance-mapping` to map audit scope to controls
3. Use `security-policy-template` to ensure policies are documented
4. Gather evidence: logs, configurations, risk assessments, incident records

Deliverables:

- Audit readiness checklist
- Evidence inventory organized by control
- Audit timeline and resource estimate
- Key metrics dashboard for auditor review

After completion, suggest follow-up commands: `assess-compliance`, `classify-data`.
