SeaClaw's UI is built with **Lit** web components. Each component uses design tokens exclusively — no hardcoded colors, sizes, or durations.

## Component Library

### Interactive Components

| Component       | Tag                    | Description                                |
| --------------- | ---------------------- | ------------------------------------------ |
| Button          | `<sc-button>`          | Primary, secondary, ghost, danger variants |
| Badge           | `<sc-badge>`           | Status indicators with semantic colors     |
| Tooltip         | `<sc-tooltip>`         | Contextual help on hover/focus             |
| Toggle          | `<sc-toggle>`          | On/off switches with micro-physics         |
| Command Palette | `<sc-command-palette>` | CMD+K global search                        |

### Layout Components

| Component | Tag            | Description                        |
| --------- | -------------- | ---------------------------------- |
| Card      | `<sc-card>`    | Elevated surface with glass option |
| Sheet     | `<sc-sheet>`   | Slide-in panel from edge           |
| Modal     | `<sc-modal>`   | Centered dialog with backdrop      |
| Dialog    | `<sc-dialog>`  | Confirmation/action dialogs        |
| Sidebar   | `<sc-sidebar>` | Navigation sidebar                 |

### Display Components

| Component    | Tag                 | Description                         |
| ------------ | ------------------- | ----------------------------------- |
| Avatar       | `<sc-avatar>`       | User avatars with fallback initials |
| Floating Mic | `<sc-floating-mic>` | Voice input indicator               |

## Token Integration

Every component consumes tokens through CSS custom properties. Example from `sc-card`:

```css
:host {
  background: var(--sc-bg-surface);
  border: 1px solid var(--sc-border-subtle);
  border-radius: var(--sc-radius-lg);
  box-shadow: var(--sc-shadow-card);
  padding: var(--sc-space-lg);
}

:host(:hover) {
  box-shadow: var(--sc-shadow-lg);
  transition: box-shadow var(--sc-duration-normal) var(--sc-ease-default);
}
```

## Component Variants

### Button Variants

```html
<sc-button variant="primary">Save</sc-button>
<sc-button variant="secondary">Cancel</sc-button>
<sc-button variant="ghost">More info</sc-button>
<sc-button variant="danger">Delete</sc-button>
```

- **Primary**: Accent background, on-accent text, gradient overlay, physics press
- **Secondary**: Surface background, accent text, subtle border
- **Ghost**: Transparent background, accent text, hover overlay
- **Danger**: Error background, on-error text

### Badge Variants

```html
<sc-badge variant="success">Online</sc-badge>
<sc-badge variant="warning">Pending</sc-badge>
<sc-badge variant="error">Failed</sc-badge>
<sc-badge variant="info">New</sc-badge>
```

## Accessibility Patterns

All components implement:

- **Focus management**: Visible focus rings using `--sc-focus-ring` token
- **Keyboard navigation**: Full keyboard support (Enter, Space, Escape, Arrow keys)
- **ARIA attributes**: Proper roles, states, and properties
- **Screen reader text**: Hidden labels where visual indicators are insufficient

### Focus Ring Standard

```css
:host(:focus-visible) {
  outline: var(--sc-focus-ring-width) solid var(--sc-focus-ring);
  outline-offset: var(--sc-focus-ring-offset);
}
```

## Motion Patterns

Components apply micro-physics contextually:

| Component            | Interaction | Physics Preset                    |
| -------------------- | ----------- | --------------------------------- |
| `sc-button`          | Click       | `button-press` / `button-release` |
| `sc-card`            | Hover       | `card-hover`                      |
| `sc-modal`           | Open        | `modal-anticipation`              |
| `sc-toggle`          | Toggle      | `toggle-squash`                   |
| `sc-command-palette` | Open        | `dropdown-anticipation`           |

## Glass Integration

Components can opt into glass tiers:

```html
<sc-card class="glass">Standard glass card</sc-card>
<sc-card class="glass-prominent">Prominent glass card</sc-card>
```
