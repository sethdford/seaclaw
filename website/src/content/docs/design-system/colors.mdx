SeaClaw's color system is built on an **ocean-aligned palette** with three accent families and three theme modes.

## Philosophy

Colors in SeaClaw evoke the ocean — deep navy backgrounds, teal accents like sunlit water, amber highlights like sandy shores, and indigo depths. Every color is purposeful:

- **Teal** — primary actions, links, focus rings
- **Amber** — secondary emphasis, warnings, warmth
- **Indigo** — tertiary accents, tags, decorative elements

## Theme Modes

### Dark Theme (Deep Ocean)

Default theme. Navy backgrounds with high-luminance text. Accent colors are bright and saturated for maximum visibility against dark surfaces.

### Light Theme (Sandy Shore)

Light backgrounds with deep text. Accent colors are slightly desaturated to maintain contrast ratios on bright surfaces. All text/background pairs meet WCAG AA (4.5:1 minimum).

### High Contrast

Maximum contrast for accessibility. Backgrounds are pure black or white, text uses extreme values. All pairs exceed WCAG AAA (7:1 minimum). Focus rings use maximum contrast.

## Contrast Requirements

All semantic color pairings are validated:

| Pairing                     | Minimum Ratio | Standard |
| --------------------------- | :-----------: | -------- |
| Text on background          |     4.5:1     | WCAG AA  |
| Large text on background    |      3:1      | WCAG AA  |
| UI components on background |      3:1      | WCAG AA  |
| High-contrast text          |      7:1      | WCAG AAA |
| Focus ring on background    |      3:1      | WCAG 2.2 |

## Usage

### CSS

```css
.card {
  background: var(--sc-bg-surface);
  color: var(--sc-text);
  border: 1px solid var(--sc-border-subtle);
}

.badge-success {
  background: var(--sc-success-dim);
  color: var(--sc-success);
}
```

### Swift

```swift
Text("Hello")
  .foregroundColor(SCTokens.accent)
  .background(SCTokens.bgSurface)
```

### Kotlin

```kotlin
Text(
  text = "Hello",
  color = SCTokens.accent,
  modifier = Modifier.background(SCTokens.bgSurface)
)
```

### C (Terminal)

```c
printf(SC_COLOR_ACCENT "Connected" SC_COLOR_RESET "\n");
printf(SC_COLOR_ERROR_TC "Error: %s" SC_COLOR_RESET "\n", msg);
```
