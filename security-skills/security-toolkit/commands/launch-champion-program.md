# Launch Champion Program Command

Chain these steps:

1. Use `security-champion-program` to design program structure, roles, and responsibilities
2. Use `security-awareness-training` to design champion training curriculum
3. Create feedback loops and communication channels
4. Design incentive structure and recognition program

Deliverables:

- Security champion program charter and objectives
- Champion role definition and responsibilities
- Training curriculum and schedule
- Feedback mechanisms and escalation paths
- Incentive and recognition program
- Champion metrics and program ROI tracking

After completion, suggest follow-up commands: `design-training`, `run-tabletop`.
