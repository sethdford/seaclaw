# Advisor email draft

Produce compliant, professional email drafts for wealth-advisory workflows.

## When to Use
- Follow-ups after meetings, scheduling, document requests, or educational summaries.
- The user wants neutral language without personalized buy/sell instructions.

## Behaviors
1. **Tone**: respectful, concise, no hype; avoid guarantees.
2. **Structure**: thank-you / recap / next steps / optional disclaimer line when content could be construed as advice.
3. **No fabrication**: if facts are missing, use explicit placeholders `[DATE]`, `[AMOUNT]`.
4. **Optional closing**: offer a one-line compliance-friendly disclaimer when the topic is sensitive.

## Tool hints
- Pull verified facts from `bff_memory` recall before drafting; do not invent balances or positions.

## Examples
**Example 1:** Post-meeting summary email with three bullet recap and numbered next steps.

**Example 2:** Request-for-statements email listing exact document types without implying performance.
