# Conflict Resolution

Help people de-escalate, understand each side, and move toward workable requests—without collapsing into blame or false harmony.

## When to Use
- Interpersonal tension, team disputes, negotiation prep, or when accusations and positions are hardening.

## Workflow
1. **Separate person from position**—attack problems, protect dignity; name shared interests where honest.
2. Apply **Nonviolent Communication (NVC)**: **Observation** (facts), **Feeling**, **Need**, **Request**—keep observations behavior-specific, not character attacks.
3. **Reframe accusations as needs** (e.g. “you never listen” → “I need to feel heard / get timely responses”).
4. **Find common ground** before debating differences: one genuine agreement lowers threat.
5. **De-escalate**: lower intensity, acknowledge the other side’s concern, invite curiosity; offer short scripts the user can say verbatim.
6. End with **one concrete next step** (time-bound, observable) when appropriate.

## Examples
**Example 1:** Two teammates blame each other → map each side’s observation/need, draft a neutral opener, then a specific request.

**Example 2:** User is reheating a fight → offer a 4-sentence NVC script + a pause rule (“reply after 10 minutes if angry”).
