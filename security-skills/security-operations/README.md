# Security Operations Plugin

Security operations toolkit including SIEM rule design, detection engineering, vulnerability management, security monitoring, log analysis, and threat intelligence.

## Skills

### 1. siem-rule-design

Design and deploy SIEM rules to detect security events, anomalies, and attack patterns.

### 2. detection-engineering

Build detection logic for common attacks using log analysis, behavioral analytics, and threat intelligence.

### 3. vulnerability-management-program

Establish vulnerability management program including scanning, prioritization, patching, and tracking.

### 4. security-monitoring-strategy

Design security monitoring strategy including log collection, centralization, alerting, and SOC staffing.

### 5. log-analysis-security

Analyze logs to investigate security events, reconstruct timelines, and identify anomalies.

### 6. alert-triage

Triage security alerts to separate signal from noise and prioritize investigation.

### 7. threat-intelligence-integration

Integrate threat intelligence feeds into detection systems and incident response processes.

## Commands

### design-detections

Design and implement SIEM rules and detection logic for priority threats.

### manage-vulnerabilities

Establish vulnerability management program with scanning, prioritization, and tracking.

### setup-monitoring

Design and deploy security monitoring infrastructure including SIEM, logging, and alerting.

## Quick Start

1. **Detection engineering**: Use `design-detections` to create SIEM rules for priority threats
2. **Vulnerability management**: Use `manage-vulnerabilities` to establish vulnerability program
3. **Monitoring infrastructure**: Use `setup-monitoring` to design SOC infrastructure
