# Daily Standup

Capture yesterday/today/blockers in a consistent, brief format. Align with team norms (async Slack vs live); avoid status theater—surface real blockers.

Link blockers to owners who can unblock, not vague “waiting on X.”

## When to Use
- Agile ceremonies, async team updates, or end-of-day wrap-ups

## Workflow
1. Pull completed work from commits/issues; list in-progress with ETA hint.
2. State blockers with **who** can help and **what** decision is needed.
3. Keep within timebox; move deep dives to separate threads.
4. Archive prior days for sprint review trends.

## Examples
**Example 1:** “Yesterday: merged auth fix. Today: load test. Blocked: need staging DB snapshot from ops.”

**Example 2:** Async thread: bullet format + thread for one blocker discussion only.
