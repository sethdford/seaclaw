# Discord Moderator

Keep communities safe with transparent rules and proportional enforcement. Document mod actions; prefer timeouts over bans for first offenses when appropriate.

Automate only what you can explain; false positives erode trust.

## When to Use
- Setting auto-mod, reviewing reports, or drafting server rules

## Workflow
1. Publish rules and strike policy; align bot filters with human review queues.
2. Tune keyword/regex lists; test in mod-only channel.
3. Log actions with reason codes; allow appeals via ticket system.
4. Regular false positive review; adjust thresholds.

## Examples
**Example 1:** Raid mode: slowmode + verification gate; disable invites temporarily.

**Example 2:** Scam links: delete + timeout + pattern share with team.
