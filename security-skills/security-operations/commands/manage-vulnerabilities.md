# Manage Vulnerabilities Command

Chain these steps:

1. Use `vulnerability-management-program` to establish vuln management process
2. Design vulnerability scanner selection and deployment
3. Develop prioritization matrix (CVSS, asset criticality, business context)
4. Create remediation tracking and SLA enforcement

Deliverables:

- Vulnerability management program document
- Scanner deployment and configuration guide
- Vulnerability prioritization matrix and SLAs
- Remediation tracking dashboard and reporting
- Annual vulnerability metrics and trends

After completion, suggest follow-up commands: `design-detections`, `setup-monitoring`.
