# Compliance & Governance Plugin

Regulatory compliance and governance toolkit including compliance mapping, security policies, audit readiness, and framework implementations (GDPR, SOC2, PCI-DSS, ISO 27001).

## Skills

### 1. compliance-mapping

Map organizational controls to regulatory frameworks (NIST CSF, ISO 27001, SOC 2, GDPR, HIPAA, PCI-DSS).

### 2. security-policy-template

Create comprehensive security policies (acceptable use, data handling, incident response, access control).

### 3. audit-preparation

Prepare for internal and external security audits including evidence gathering and remediation tracking.

### 4. gdpr-assessment

Assess GDPR compliance including data inventory, DPA agreements, privacy by design, and breach notification.

### 5. soc2-controls

Implement SOC 2 Type II controls (availability, processing integrity, confidentiality, security, privacy).

### 6. pci-dss-review

Review and implement PCI-DSS compliance for payment card data handling.

### 7. data-classification

Classify organizational data by sensitivity level and apply appropriate protections.

### 8. privacy-impact-assessment

Conduct Privacy Impact Assessments (PIAs) for new systems processing personal data.

## Commands

### assess-compliance

Assess organizational compliance against key regulatory frameworks.

### prepare-audit

Prepare for security audits through evidence gathering and control verification.

### classify-data

Conduct data classification and define protection requirements.

## Quick Start

1. **Compliance assessment**: Use `assess-compliance` to map controls to frameworks
2. **Audit preparation**: Use `prepare-audit` to gather evidence and prepare for audits
3. **Data classification**: Use `classify-data` to classify organizational data
