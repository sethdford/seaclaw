# Security Toolkit Plugin

Cross-cutting security utilities including security culture, training, architecture reviews, bug bounty programs, metrics, red team exercises, and security documentation.

## Skills

### 1. security-champion-program

Establish security champion programs to build security culture, enable developers, and create feedback loops.

### 2. security-awareness-training

Design and implement security awareness training for employees covering phishing, social engineering, password hygiene, and data handling.

### 3. secure-architecture-review

Conduct secure architecture reviews to identify design-level security issues before implementation.

### 4. bug-bounty-program

Establish and manage bug bounty programs to incentivize external security research and responsible disclosure.

### 5. security-metrics-dashboard

Build security metrics dashboards to track KPIs (MTTR, patch time, detection rate) and demonstrate security ROI.

### 6. red-team-exercise

Plan and execute red team exercises (simulated attacks, tabletop scenarios) to test defenses and response capability.

### 7. security-documentation

Create and maintain security documentation including design docs, incident playbooks, and runbooks.

## Commands

### launch-champion-program

Establish security champion program with training, feedback channels, and incentives.

### design-training

Design and deliver security awareness training program for employees.

### run-tabletop

Plan and execute tabletop exercises to test incident response procedures and team readiness.

## Quick Start

1. **Build security culture**: Use `launch-champion-program` to establish champion program
2. **Awareness training**: Use `design-training` to implement awareness training
3. **Incident preparedness**: Use `run-tabletop` to conduct tabletop exercises
