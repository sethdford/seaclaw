# Report Generator

Produce reports that answer the decision at hand. Lead with conclusions; support with charts/tables that respect accessibility (not color-only).

Automate data refresh; version the report template with the query logic.

## When to Use
- Executive summaries, weekly ops reports, or compliance packs

## Workflow
1. Define audience question and success metrics.
2. Query/transform data; validate totals against source-of-truth dashboards.
3. Choose viz types per data shape; label axes; note caveats.
4. Export PDF/HTML; store parameters used for reproducibility.

## Examples
**Example 1:** SLA report: error budget consumed, top incidents, customer impact table.

**Example 2:** Growth report: cohort retention curves with definition footnotes.
