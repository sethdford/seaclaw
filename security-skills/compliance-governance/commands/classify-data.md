# Classify Data Command

Chain these steps:

1. Use `data-classification` to classify all organizational data
2. Use `privacy-impact-assessment` for data with personal information
3. Map classification levels to protection requirements (encryption, access control, retention)

Deliverables:

- Data classification framework (Public, Internal, Confidential, Restricted)
- Data inventory with classifications
- Protection requirements matrix by classification level
- Data handling, storage, and retention policies

After completion, suggest follow-up commands: `assess-compliance`, `prepare-audit`.
