# Setup Monitoring Command

Chain these steps:

1. Use `security-monitoring-strategy` to design SOC and monitoring architecture
2. Use `siem-rule-design` to configure SIEM rules and alerting
3. Use `log-analysis-security` to design log retention and analysis procedures
4. Use `alert-triage` to design alert triage and escalation procedures

Deliverables:

- Security monitoring architecture and data flow
- SIEM configuration and rule set
- Log retention policy and storage plan
- Alert triage procedures and escalation matrix
- SOC staffing and shift requirements
- Monitoring metrics and dashboard (MTTR, alert accuracy, detection coverage)

After completion, suggest follow-up commands: `design-detections`, `manage-vulnerabilities`.
